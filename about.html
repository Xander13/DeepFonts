Deep font

Deep font is a ML based font creation app. The ML learned to understand what each fonts in the library and converts the data into a jason formats. When users add two different fonts together, it pools the data can crate a new fonts based on users linking.

Instructions

❮ ❯

Insert


Add png files of different typeface you wish to add. You can add up to 4 different typographical images for the ML to analuzed and recreate your taste.


The file should be no larger than 1920 X 1080.

Watch


Take a step back and watch the ML create the letter forms you created, You can reorder the list to priotize what font face you like and it will change the font to your liking.

Print


You can export your custom font to web font or font file to use on your next project. Click save to save in your downloads and you will have a zip file of your custom font you have created.
